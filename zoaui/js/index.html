JavaScript

ZoaUI requires jQuery (1.8 or above) to work.

JavaScript APIs can be accessed via HTML attributes, (and as of v0.1.4, only so), which are prefixed with data-zo- (see corresponding documentation pages).

Because event listeners are bound to the document, it works on elements dynamically attached to the DOM as well.
