项目名称: 心愿墙
1) 推荐浏览所用平台: Windows 8+ / Server 2012+ (因为字体支持关系)
2) 浏览器支持: IE9+ (IE9 无法看到部分效果, 10 或以上完全支持), Firefox 最新若干版本, Chrome 最新若干版本
3) 分辨率: 采用响应式设计，可以兼容几乎全部分辨率
4) 这 3 个愿望是 demo 用，邮寄心愿请附上图片，手动更新请按照此格式
                    <li>
                        <a href="[图片裁切为 16:9 的 URL]" data-width="[图片裁切后宽度 (缩放时要用)]" data-height="[图片裁切后高度]">
                            <img src="[图片裁切为 16:9 的 URL]" alt="[图片替代文本]" />
                            <div class="gallery-caption">
                                <p>[心愿内容]</p>
                                <p>&mdash; [许愿者姓名, 若无本段可以省略]</p>
                            </div>
                        </a>
                    </li>
若不附上图片，我有极小概率帮你选择图片。
5) 在受支持的浏览器下，单击愿望图片可以平滑缩放查看大图 (IE9 由于不支持 transition 无法平滑缩放)
6) 中文支持效果不太好，因为 Windows Server 2012 (本来考虑的是 Windows 7) 木有 Microsoft YaHei Light
7) 木有后端运行环境，大家放心
